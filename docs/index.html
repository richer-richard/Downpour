Release shape

Source-first, local build, no binary download

This release is a build-from-source installation for macOS. It does not ship a notarized .app, .zip, or .dmg. Download the tagged source, run ./install.sh, and let the script handle the toolchain, app build, install, and local cleanup.

Install

Run the installer from the release source

git clone https://github.com/richer-richard/Downpour.git
cd Downpour
git checkout v0.1.0
./install.sh

The first build can take several minutes on a fresh machine.

What the script does

  • Checks or installs Xcode Command Line Tools, Homebrew, Node.js, pnpm, and Rust.
  • Builds Downpour.app locally from the tagged source.
  • Copies the finished app into /Applications.
  • Removes repo-local build artifacts when installation finishes.

Notes

What to expect on macOS

  • macOS only.
  • The installer may ask for administrator access when writing to /Applications.
  • If Gatekeeper warns on first launch, open the app from Finder or allow it in System Settings.
