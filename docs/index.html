Release shape

Source-first, local build, no binary download

This release is a build-from-source installation for macOS. It does not ship a notarized .app, .zip, or .dmg. Download the tagged source, run ./install.sh, and let the script handle the toolchain, app build, install, and local cleanup.

Build path

What the script actually does

  1. 01

    Provision the toolchain

    Checks or installs Xcode Command Line Tools, Homebrew, Node.js, pnpm, and Rust.

  2. 02

    Build the tagged source

    Compiles Downpour.app locally from the release checkout.

  3. 03

    Install the finished app

    Copies the application bundle into /Applications and replaces any older install.

  4. 04

    Clear repo-local build residue

    Removes the generated frontend and Tauri build artifacts after the install completes.
